# 記事が表示されない問題のトラブルシューティング

## 現在の状況

記事データは正しく保存されています：
- **記事数**: 2件
- **1件目**: 「おすすめ家庭用脱毛器比較」- コンテンツ10,579文字 ✅
- **2件目**: 「【2024年版】おすすめノートPC比較ランキング TOP5」- コンテンツ0文字 ⚠️

## 確認事項

### 1. ブラウザのコンソールを確認

ブラウザの開発者ツール（F12キー）を開き、コンソールタブで以下のログを確認してください：

```
[Affiling] 2件の記事を読み込みました
[Affiling] データベースから 2件の記事を取得しました
[Affiling] 2件の記事を表示します
[Affiling] 2件の記事をレンダリングします
```

エラーメッセージがないか確認してください。

### 2. ネットワークタブを確認

1. ブラウザの開発者ツールを開く
2. 「Network」タブを選択
3. ページをリロード
4. `data/affiling_articles.json` が正しく読み込まれているか確認
   - ステータスコードが200であること
   - レスポンスに記事データが含まれていること

### 3. 記事がフィルタリングで除外されていないか確認

記事リストページで：
- フィルターが「すべて」になっているか確認
- 検索ボックスにキーワードが入力されていないか確認

### 4. 記事データの確認

記事データは以下のコマンドで確認できます：

```bash
python3 -c "
import json
with open('data/affiling_articles.json', 'r', encoding='utf-8') as f:
    articles = json.load(f)
    for i, article in enumerate(articles, 1):
        print(f'記事{i}: {article.get(\"title\", \"なし\")}')
        print(f'  ID: {article.get(\"id\", \"なし\")}')
        print(f'  カテゴリ: {article.get(\"category\", \"なし\")}')
        print(f'  日付: {article.get(\"date\", \"なし\")}')
        print()
"
```

## 解決方法

### 記事が表示されない場合

1. **ブラウザのキャッシュをクリア**
   - Ctrl+Shift+Delete（Windows/Linux）
   - Cmd+Shift+Delete（Mac）
   - 「キャッシュされた画像とファイル」を選択して削除

2. **ページを強制リロード**
   - Ctrl+F5（Windows/Linux）
   - Cmd+Shift+R（Mac）

3. **JSONファイルのパスを確認**
   - `affiling.html` と同じ階層に `data/affiling_articles.json` が存在することを確認

4. **記事同期を再実行**
   ```bash
   export NOTION_API_TOKEN="your_token"
   python3 scripts/sync_affiling_articles.py pull --output data/affiling_articles.json
   ```

### 2件目の記事のコンテンツが空の場合

2件目の記事「【2024年版】おすすめノートPC比較ランキング TOP5」は、ページ本文が空のためコンテンツが取得できていません。

**解決方法：**
1. Notionで記事を開く
2. ContentプロパティまたはCommentプロパティにあった記事内容を、ページ本文に移動してください
3. 記事同期を再実行してください

## よくある問題

### 記事が1件も表示されない

- JSONファイルが読み込めていない可能性があります
- ブラウザのコンソールでエラーを確認してください
- JSONファイルのパスが正しいか確認してください

### 記事が部分的に表示されない

- フィルターや検索で記事が除外されている可能性があります
- ブラウザのコンソールでフィルタリングのログを確認してください

### 記事詳細ページでコンテンツが表示されない

- 記事のコンテンツが空（0文字）の場合、「記事の内容を読み込めませんでした」と表示されます
- Notionのページ本文に記事内容を記入してください
