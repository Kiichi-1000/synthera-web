# Notionデータベースプロパティ削除ガイド

## 概要
Affiling記事の記事内容は、**ページ本文（ブロック）**から取得するように変更されました。そのため、以下のプロパティは不要になりました：

- **Content** プロパティ
- **Comment** プロパティ

これらのプロパティは、Notionデータベースから手動で削除する必要があります。

## プロパティ削除手順

### 1. Notionデータベースを開く
Affiling記事のデータベースをNotionで開きます。

### 2. プロパティを削除
1. データベースの右上にある「...」（三点メニュー）をクリック
2. 「Properties」を選択
3. 以下のプロパティを削除：
   - **Content** プロパティ
   - **Comment** プロパティ

各プロパティの右側にある「...」をクリックし、「Delete」を選択して削除します。

### 3. 確認
削除後、データベースに以下のプロパティのみが残っていることを確認してください：

**必須プロパティ:**
- Title (タイトル)
- Status (ステータス)
- Excerpt (概要)
- Category (カテゴリ)
- Date (日付)
- Image (画像)
- Read Time (読了時間)
- Product Count (商品数)
- Tags (タグ)

## 記事内容の保存場所

記事内容は、Notionページの**本文（ブロック）**に直接記入してください。

### ページ本文への記入方法
1. データベースの記事を開く
2. ページ内に記事内容を記入
3. 見出し、段落、リスト、画像などのブロックを使って記事を作成

記事内容は自動的にページ本文から取得され、HTMLに変換されてサイトに表示されます。

## 注意事項

- プロパティを削除する前に、重要なデータがないか確認してください
- 既存の記事は、ページ本文に記事内容があることを確認してください
- Contentプロパティに記事内容がある記事は、ページ本文に移動する必要があります

## トラブルシューティング

### 記事が表示されない場合
1. 記事のページ本文に内容が記入されているか確認してください
2. 記事同期スクリプトを実行して、最新の内容を取得してください

### プロパティが削除できない場合
- プロパティが他の場所で使用されている可能性があります
- データベースの権限を確認してください
