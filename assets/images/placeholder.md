# Images Directory

このディレクトリには、ウェブサイトで使用する画像ファイルを配置してください。

## 推奨画像サイズ

- **ヒーロー画像**: 1920x1080px
- **プロジェクト画像**: 800x600px
- **記事画像**: 600x400px
- **アイコン**: 32x32px または 64x64px

## 画像形式

- **写真**: JPEG, WebP
- **アイコン**: SVG, PNG
- **ロゴ**: SVG, PNG

## 最適化

画像は以下のツールで最適化することを推奨します：
- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)
- [ImageOptim](https://imageoptim.com/)
